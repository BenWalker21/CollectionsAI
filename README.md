# CollectionsAI

CollectionsAI is a static marketing homepage for an AI collections copilot.

The page presents the product concept around:

- AI prioritization for overdue invoices
- Smart collections queues
- Personalized email follow-ups
- Promise-to-pay tracking
- Collections analytics and recovered cash reporting

## Local preview

```bash
npm run serve
```

Then open `http://localhost:4173`.

## Checks

```bash
npm run check
```
